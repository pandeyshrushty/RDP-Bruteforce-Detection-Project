**RDP Brute Force Detection Project

(Windows 10 + Parrot OS + Event Logs)**

This project demonstrates how failed Remote Desktop Protocol (RDP) login attempts can be detected using Windows Event Logs.
A controlled brute-force attack was generated using Hydra from Parrot OS, and the resulting failed login logs (Event ID 4625) were collected and analyzed.
